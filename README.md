# 🚀 100 Days of DevOps

Welcome to my **100 Days of DevOps Challenge**.

This repository documents my journey of learning DevOps, cloud technologies, automation, infrastructure, and modern software delivery practices over the next 100 days.

The goal is not only to learn new concepts but also to build projects, solve real-world problems, and develop hands-on experience through consistent daily practice.

---

## 🎯 Objectives

- Build a strong foundation in DevOps principles
- Learn infrastructure and system administration concepts
- Improve automation and scripting skills
- Understand networking fundamentals
- Explore containerization and orchestration
- Learn CI/CD practices
- Gain cloud and infrastructure knowledge
- Work on practical projects and challenges
- Document progress and share learnings

---


## 📚 Learning Areas

Throughout this challenge, I may explore topics including:

- Linux & System Administration
- Networking Fundamentals
- Shell Scripting & Automation
- Version Control
- Containers & Virtualization
- CI/CD
- Infrastructure as Code
- Monitoring & Observability
- Cloud Platforms
- Security Fundamentals
- DevOps Tools & Workflows
- Real-World Projects
